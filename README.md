# eGov Parser

Скрипт для парсинга и поиска онлайн-сервисов с портала [eGov.kz](https://egov.kz/cms/ru/online-services/for_citizen).

## Описание

Данный скрипт предназначен для:
- Автоматического сбора списка онлайн-сервисов с портала eGov.kz
- Сохранения данных в JSON формате
- Предоставления удобного поискового интерфейса по сохраненным сервисам

## Требования

- Python 3.6 или выше
- Модули Python:
  - requests
  - beautifulsoup4

## Установка

Установите необходимые зависимости с помощью pip:

```bash
pip install requests beautifulsoup4
```

## Запуск

1. Скачайте файл скрипта `scrap.py`
2. Убедитесь в наличии интернет-соединения
3. Запустите скрипт:

```bash
python scrap.py
```

## Использование

### Сохранение данных

При запуске скрипт автоматически:
- Загружает список сервисов с eGov.kz
- Сохраняет данные в файл `egov_services.json` в текущей директории

Формат сохраняемых данных:

```json
[
  {
    "name": "Название сервиса",
    "link": "Ссылка на сервис"
  }
]
```

### Поиск сервисов

После сохранения данных вы можете искать сервисы:

1. Введите поисковый запрос (например: "налоги")
2. Просмотрите найденные результаты
3. Для выхода введите "пока"

Пример работы:

```text
ищи что надо (напиши "пока" чтобы выйти отсюда): налоги

нашлось 2 штук:
1. Оплата налогов -> https://egov.kz/cms/ru/online-services/for_citizen/nalogi
2. Проверка налоговой задолженности -> https://egov.kz/cms/ru/online-services/for_citizen/proverka-nalogov
```

## Обработка ошибок

- При недоступности сайта: выводится сообщение "не нашлось"
- При отсутствии результатов поиска: "ничего нет, давай по новой"

## Решение проблем

1. Если скрипт не может получить данные:
   - Проверьте доступность сайта eGov.kz
   - Убедитесь в наличии интернет-соединения
   - Попробуйте запустить скрипт позже

2. Если поиск не дает результатов:
   - Попробуйте использовать другие ключевые слова
   - Проверьте правильность написания запроса
